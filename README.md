# Сервис парсинга и отображения новостей RSS-лент Ведомостей

## Требования:
1. NodeJS поддерживаемой версии
2. Express.JS и EJS (устаналивается полу-автоматически, см. пункт 2)

## Установка:
1. Клонируйте (установите) этот репозиторий
2. Установите зависимости - npm install

## Использование:
1. Запустите сервер (script.js). Примечание: В случае успешного запуска в консоли отобразится сообщение "Готов к работе..."
2. Откройте в браузере маршрут, по которому работает сервер.

### Пример использования:
1. node script.js - запуск сервера
2. http://localhost:3000/1/news/for/auto, где 1 это количество отображаемых новостей, а auto это категория новостей.

## Формат запроса
http://localhost:3000/:number/news/for/:category

### Параметры запроса
1. :number - количество отображаемых новостей
2. :category - категория отображаемых новостей

## Категории новостей
Все категории доступны по ссылке: https://www.vedomosti.ru/info/rss
